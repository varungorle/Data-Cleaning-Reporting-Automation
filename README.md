# 🏥 Data Cleaning & Reporting Automation (Healthcare Dataset)

## 📌 Project Overview
This project automates the process of data cleaning, preprocessing, and reporting using a healthcare dataset. It handles missing values, removes duplicates, standardizes data, and generates automated reports with visual insights. The goal is to improve efficiency in data preprocessing and enable quick decision-making through automated reporting.

---

## 🚀 Features
- Automated dataset loading
- Data cleaning (missing values, duplicates, formatting)
- Standardization of text and column names
- Automated statistical report generation
- Data visualization using charts
- Cleaned dataset export for further use
